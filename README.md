# 🚀 CVForge – AI Resume Builder

CVForge is an AI-powered resume builder web application that helps users create professional, ATS-optimized resumes and cover letters in minutes.

Built using a modern AI app builder (Base44), CVForge focuses on simplifying resume creation with intelligent suggestions, ATS scoring, and clean UI templates.

---

## ✨ Features

- 🤖 AI-powered resume generation
- 📊 ATS (Applicant Tracking System) score checker
- 🧠 Smart resume improvement suggestions
- 📄 Cover letter generator
- 🎨 Modern UI with responsive design
- ⚡ Fast and interactive dashboard
- 💾 Resume storage and management

---

## 🛠️ Tech Stack

- React.js (Frontend)
- Tailwind CSS / UI Components
- Base44 AI Platform
- JavaScript (ES6+)
- Node-style modular architecture
